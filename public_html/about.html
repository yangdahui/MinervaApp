Schematic of the DWM framework

The core of DWM includes: (1) a cloud-based data collection component for extracting, transforming, and loading heterogeneous data, (2) a comprehensive information extraction component for extracting HSCB relevant information in unstructured text, (3) a machine learning and modeling component for HSCB analysis, (4) a data warehouse component that supports queries of structured data, unstructured data and analysis results, (5) a query and visualization component for accessing HSCB data and features. Each component consists of a set of cloud computing services. These services can coordinate with each other for efficient information flow.
